## Структура проекта

- `index.html` - Главная страница (Bootstrap версия)
- `index_v1.html` - Главная страница (оригинальная версия)
- `goods.html` - Страница товаров (Bootstrap версия)  
- `goods_v1.html` - Страница товаров (оригинальная версия)
- `news.html` - Новости
- `practices.html` - Практики
- `contacts.html` - Контакты
- `style/styles.css` - Стили для оригинальных версий

## Особенности Bootstrap версий

### Главная страница (index.html)
- Полностью построена на компонентах Bootstrap
- Адаптивная навигация
- Карточки с hover-эффектами
- Секция с навыками в виде бейджей
- Статистика в виде счетчиков
- Отзывчивый дизайн

### Страница товаров (goods.html)
- Сетка товаров на Bootstrap Grid
- Фильтры товаров
- Карточки с метками скидок
- Интерактивная корзина
- Toast-уведомления
- Адаптивные изображения

## Технологии

- HTML5
- CSS3
- Bootstrap 5.3.2
- Bootstrap Icons
- JavaScript

## Установка

Просто откройте нужный HTML файл в браузере. Bootstrap подключается через CDN.
